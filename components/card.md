# Card

Tarjetas para agrupar información relacionada en bloques de lectura fácil.

## Principios

- Bordes sutiles, sin sombras pesadas.
- Úsalas para modular contenido: métricas, resúmenes, listas cortas.
- Evita convertirlas en “cajas” decorativas sin jerarquía.

## Anatomía

- `title` (opcional)
- `body` (contenido)
- `footer` (opcional, acciones o metadatos)

## Variantes

- Base: para contenido estándar.
- Elevated: para contenido destacado o seleccionado.
