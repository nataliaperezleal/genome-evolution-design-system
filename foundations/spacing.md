# Spacing — Genome Plus DS

## Escala de espaciado
| Token | Valor |
|---|---|
| `space/2` | 2px |
| `space/4` | 4px |
| `space/8` | 8px |
| `space/16` | 16px |
| `gap/sm` | 8px |
| `gap/xs` | 4px |

## Uso por componente
- **Accordion MD**: padding H `space/16`, V derivado de size/40
- **Accordion SM**: padding H `space/8`, V derivado de size/32
- **Button MD**: padding H 16px, V 8px
- **Button SM**: padding H 12px, V 4px
- **Table header**: padding H `space/16`, V `space/8`
- **Table cell small**: padding H `space/16`, V `space/4`
- **Table cell medium**: padding H `space/16`, V `space/8`
